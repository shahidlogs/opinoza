* withdrawal modal / form

* **PayPal**

* show Account Title in admin withdrawal details/list where appropriate

Behavior by method:

* **PayPal**: only require the PayPal-relevant payment detail field
* **Bank Transfer**: require:

  * account number / IBAN

Example:

* PayPal → placeholder like `PayPal email`
* Bank Transfer → placeholders like `Account number / IBAN` and `Account title`
